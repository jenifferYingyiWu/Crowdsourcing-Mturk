Specify Data File

  • The file must be in CSV format.
  • The first line must contain the names of the columns.
  • If a column contains image links, an asterisk must be placed immediately after the column name.
  • Image links must be:
    • relative, if the images are stored on this web server.
    • absolute, if the images are stored on an external web server.

Specify Question Format File

  • It must be an XML file conforming to the QuestionForm schema.
  • If ${column_name} is found within a field, it will be replaced by the contents of the specified column in the actual HIT.
  • The answer must be in the form of one radio button question where:
    • The <SelectionIdentifier> fields contain unique strings representing the actual answers in the <Text> field.
    • Only 1 answer can be chosen (min=1, max=1).

Specify Data File

  • The file must be in CSV format.
  • The first line must contain the names of the columns.
  • Image links must be valid HTTP URLs.

Specify Question Format File

  • It must be an XML file conforming to the QuestionForm schema.
  • If ${column_name} is found within a field, it will be replaced by the contents of the specified column in the actual HIT.
  • The answer must be in the form of one radio button question where:
    • The <Text> fields contain the actual values for the labels.
    • The <SelectionIdentifier> fields contain contain unique strings representing these actual values.
    • Only 1 radio button can be selected as the answer.

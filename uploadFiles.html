Specify Data File

  • The file is in CSV format.
  • The first line contains the names of the columns.
  • Image links are valid HTTP URLs.

Specify Question Format File

  • The file is in QuestionForm format.
  • If ${columnName} is found within a field, it will be replaced by the contents of the specified column in the actual HIT.
  • The AnswerSpecification field contains the SelectionAnswer field.
  • For the Selection fields within the SelectionAnswer field:
    • The Text fields contain the actual values for the labels.
    • The SelectionIdentifier fields contain unique abbreviations representing these actual values.
    • MinSelectionCount and MaxSelectionCount have a value of 1.
